Datos abiertos

Hemos decidido abrir algunos de nuestros bancos de datos sobre la tos COVID-19.

Queremos beneficiar humanidad y decidimos ser los pioneros en abrir el primer banco de datos de COVID-19 y hacerlo gratis.

La pandemia de COVID-19 es extremadamente dañina a la sociedad y necesitamos trabajar juntos a superarlo. La IA necesita muchos data para funcionar, y no tenemos conocimiento de otros datos abiertos para el conjunto de datos de COVID-19 en cualquier parte del mundo. Queremos beneficiar humanidad y decidimos ser los pioneros en abrir el primer conjunto de datos de COVID-19 y hacerlo gratis. Esto formentará innovación y colaboración del sector académico y industrial en todo el mundo.

Fuente de datos

Nuestros datos son buena calidad porque fueron colectados en un hospital bajo la direccion de médicos cumpliendo procedimientos normalizados de trabajo. Nuestros datos son pre-procesados y etiquetados con COVID-19, (adquiridos de las pruebas de PCR), junto con las demográficas del paciente (edad, género, historia médica).

Empieza

Nuestros datos y instrucciones de uso se pueden encontrar en nuestra página de Github. Nuestros modelos de tos y texto para processar los datos estan siendo subidos.

Colaboración

Nos complace la revisión de nuestros datos y código. Esperamos crear una comunidad de investigadores impulsados a utilizar estos datos para crear soluciones para la pandemia. Por favor, envíanos un correo electrónico a open-data@virufy.org si estás interesado.
